# 🌸 Kawaii Todo List

A responsive **Kawaii Todo List** user interface built using **HTML5** and **CSS3** based on the provided Figma design. The goal of this project was to recreate the original design as accurately as possible by matching its layout, colors, typography, spacing, rounded corners, and overall kawaii aesthetic. This project focuses on the visual design only, without implementing any JavaScript functionality.

The interface includes a rounded input field with an add button, a list of sample tasks with edit and delete icons, a soft pink color palette, a faded bunny background illustration, and a cute bunny mascot at the bottom. The layout is fully responsive and adapts to different screen sizes using CSS media queries.

## 💡 Challenges

One of the biggest challenges was matching the original Figma design as closely as possible. Small details such as spacing, border radius, font sizes, button positioning, and the faded bunny watermark required careful adjustments. Achieving a responsive layout while preserving the overall appearance on smaller screens also required several refinements using CSS media queries.
